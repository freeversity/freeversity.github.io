# Как отцентровать элемент?

Чтобы отцентровать _блочный_ элемент, нужно выполнить следующие действия:

1. Задать элементу ширину, которая меньше ширины родительского контейнера.
2. Задать для внешних отступов справа и слева значение `auto`.

Примеры кода:

```css
selector {
  width: 100px;
  margin: 0 auto;
}

selector {
  width: 100px;
  margin-left: auto;
  margin-right: auto;
}
```
