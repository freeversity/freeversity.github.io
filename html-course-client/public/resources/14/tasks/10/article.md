# Псевдокласс `:nth-child`

Псевдоклассы из предыдущего примера относятся к семейству псевдоклассов, помогающих выбирать элементы по их расположению.

Вспомним [задание 4](/chapters/14/tasks/4). В нём каждому тегу `<li>` был задан собственный класс. Используя классы, мы могли выбрать любой из пяти тегов. Если бы тегов было десять, то пришлось бы использовать десять разных классов.

С помощью псевдокласса `nth-child` можно выбирать теги по порядковому номеру, _не используя классы_. Синтаксис псевдокласса: `селектор:nth-child(выражение)`. _Выражением_ может быть число или формула. Например:

```css
li:nth-child(2) { ... }
li:nth-child(4) { ... }
li:nth-child(2n) { ... }
```

Первый селектор выберет второй элемент списка, второй селектор — четвёртый элемент списка, третий селектор — все чётные элементы.
