# Смещение тени по вертикали

Еще раз синтаксис:

    box-shadow:
    [inset] — внутренняя
    5px     — смещение по x
    10px    <— смещение по y
    [2px]   — размытие
    [3px]   — растяжение
    [red];  — цвет

Второе цифровое значение задаёт _смещение по вертикали_. Оно работает аналогично предыдущему значению, смещая тень вверх или вниз.
