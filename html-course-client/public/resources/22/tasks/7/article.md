# Наложение элементов грида и свойство `z-index`

Напоминает [абсолютное позиционирование](/chapters/23/tasks/8), не правда ли?

Как и на абсолютно спозиционированные элементы, на пересекающиеся грид-элементы действует [свойство `z-index`](/chapters/23/tasks/19). Это свойство управляет тем, как перекрываются элементы. Значение этого свойства — целое число:

```css
.element {
  z-index: 2;
}
```

Чем больше значение `z-index`, тем выше элемент в «стопке». Если у двух элементов одинаковый `z-index`, то выше будет располагаться тот, который идёт следующим по разметке.
