# Иконка-трансформер, шаг 3

Отлично! Теперь с помощью трансформаций сместим и повернём короткие прямоугольники так, чтобы получилась стрелка.

Напоминаем, что к элементам или псевдоэлементам можно применить несколько трансформаций одновременно, для этого их нужно перечислить через пробел, например так:

```css
.some-class::before {
  transform: translate(15px, 15px) rotate(90deg);
}
```

Порядок следования трансформации важен, так как при разном порядке могут получаться разные результаты.
