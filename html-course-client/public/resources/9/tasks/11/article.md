# Верстаем асимметричную карточку

Сверстаем ещё одну карточку:

![Макет карточки](/resources/9/assets/grid01.svg)

Расположение элементов внутри карточки выглядит знакомо: содержимое разбито на две колонки, между рядами и колонками имеются отступы. Для такой сетки удобнее всего использовать [гриды](/chapters/7/tsaks/4).

Однако карточка немного асимметрична: заголовок шире, чем список опций и первая колонка.

![Макет карточки](/resources/9/assets/grid02.svg)

Попробуем сверстать карточку, используя уже знакомые приёмы. Сперва разделим содержимое карточки на [две колонки](/chapters/7/tsaks/5) и добавим [отступы](/chapters/7/tsaks/9).

```css
.grid-container {
  display: grid;
  grid-template-columns: 100px 200px;
  gap: 10px;
}
```

Потом c помощью свойства `width` зададим заголовку ширину больше, чем ширина колонки.
