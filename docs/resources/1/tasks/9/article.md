# Задаём стили по классу

Кекс провёл ревью текущего состояния своего сайта и остался вполне доволен. Но потом вспомнил, что ему часто звонят клиенты и спрашивают, какие есть тарифы в его магазине. А он уже устал рассказывать, что в «Эконом» входит первый пункт списка преимуществ, в «Стандарт» ещё два пункта, а в «VIP» ещё два.

И вот новая задача от босса: Оформите список преимуществ так, чтобы сразу было понятно, что в какой тариф входит. Сразу договоримся: на нас визуальное решение, на вас техническая реализация.
Сначала зададим всем элементам списка оформление по умолчанию (отступ, рамку, цвет текста), которое соответствует тарифу «Эконом». Другие тарифы оформим позже.

Как лучше это сделать? Можно, как и на прошлом шаге, добавить новое CSS-правило с селектором по тегу `li`. Решение простое, но оно затронет вообще все элементы списков на странице. А нам нужно оформить только определённые теги.

Чем отличаются элементы обычного списка и списка преимуществ? Классом! У элементов списка преимуществ есть класс `advantages-item`. То есть нам достаточно выбрать теги с этим классом. И для этого есть специальный селектор, селектор по классу, например:

```css
.advantages-item {…}
```

Селектор по классу начинается с точки, за которой следует имя класса. Кстати, в стилях уже есть CSS-правило для класса `advantages-item`, поэтому нужно просто добавить в него несколько свойств.
