# «Схлопывание» внешних отступов

Во втором блоке части разберём некоторые важные тонкости блочной модели. И начнём с внешних отступов (`margin`).

_Вертикальный_ отступ между двумя соседними элементами равен максимальному отступу между ними. Если отступ одного элемента равен `50px`, а второго `30px`, то отступ между ними будет `50px`:

![Схлопывание внешних отступов](/resources/18/assets/margin-collapse.svg)

Этот эффект называется эффектом «схлопывания» внешних отступов.

_Горизонтальные_ отступы между элементами просто складываются. Например, горизонтальный отступ между двумя элементами с отступами `30px` будет равен `60px`.
