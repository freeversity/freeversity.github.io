# Добавляем отступы между карточками

Мы обнулили отступы у списка и задали ширину и внутренние отступы карточкам. Теперь нужно расположить карточки в ряд и добавить отступы между ними. Расположить карточки в ряд проще всего с помощью [флексбокса](/chapters/7/tasks/12), потому что флекс-элементы по умолчанию выстраиваются именно так.

![Макет карусели](/resources/9/assets/flex03.jpg)

Обратите внимание, в карусели должен быть только один ряд карточек, поэтому нет необходимости разрешать [перенос на новую строку](/chapters/8/tasks/6).

Так как отступы между карточками должны быть фиксированными, используем для них свойство `margin`. Поступим так же, как в [прошлой части](/chapters/8/tasks/10): сперва добавим внешний отступ справа всем карточкам, а потом уберём его у последней карточки с помощью [псевдокласса](/chapters/8/tasks/11). Так как в карусели одновременно будут показываться только три карточки, то последней всегда будет третья карточка.

В прошлой части мы использовали псевдокласс `:nth-child`, чтобы выбрать _каждый третий_ элемент. Сейчас нам нужно убрать отступ только у одной карточки, у третьей, поэтому селектор будет выглядеть немного иначе:

```css
// Выберет третий элемент с классом item
.item:nth-child(3) { ... }
```

Сделаем список `cards-list` флекс-контейнером и добавим элементам с классом `card` внешний отступ справа. Потом с помощью псевдокласса `:nth-child` уберём лишний отступ у третьей карточки.

> ***BTW:***
> Выбрать последний элемент также можно с помощью псевдокласса `:last-child`.
