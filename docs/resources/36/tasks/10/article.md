# Логотип: нестандартный шрифт

Помимо теней, необычных текстовых эффектов можно добиться с помощью подключения нестандартных шрифтов. В [части про возможности HTML и CSS](/chapters/10/tasks/9) уже рассматривался пример подключения шрифтов с помощью конструкции `@font-face`.

В этом и следующих двух заданиях мы создадим логотип вымышленной компании «Warm photo» с использованием веб-шрифтов.

Для начала подключим шрифт для надписи логотипа и используем его в CSS.
