# Испытание: зигзаг

Используя обводки с разными параметрами из двух чёрных треугольников нужно получить зигзаг.

Толщина всех обводок кратна `10`. Используйте цвета <code style="background-color: teal; color: #fff">teal</code>, <code style="background-color: lightseagreen; color: #fff">lightseagreen</code> и <code style="background-color: gold">gold</code>.
