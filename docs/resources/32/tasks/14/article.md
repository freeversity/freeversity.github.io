# Испытание: восход солнца

Используя заданные фигуры и различные параметры обводки нарисуйте солнце с множеством лучей.

Все значения кратны `5`. Используемые цвета: <code style="background-color: tomato; color: #fff">tomato</code>, <code style="background-color: darkorange; color: #fff">darkorange</code>, <code style="background-color: gold">gold</code>.
