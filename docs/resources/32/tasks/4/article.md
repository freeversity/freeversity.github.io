# Испытание: разноцветные квадраты

Раскрасьте квадраты, используя цвета <code style="background-color: tomato; color: #fff">tomato</code>, <code style="background-color: lightseagreen; color: #fff">lightseagreen</code> и <code style="background-color: #ffd700">#ffd700</code>.

Заливка бирюзового квадрата должна быть полупрозрачной, нижний правый квадрат нужно сделать полностью прозрачным.
