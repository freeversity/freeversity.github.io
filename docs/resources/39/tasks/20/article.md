# Запуск фабрики, шаг 1

Настало время проверить полученные знания в деле и запустить производство роботов.

В этом и следующем задании для объектов уже созданы кадры анимации, не хватает только нужных анимационных свойств.

Начнём с анимации спутниковой тарелки и самолёта, который доставит нужный для производства груз.
