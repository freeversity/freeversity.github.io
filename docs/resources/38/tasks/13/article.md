# Чекбоксы, шаг 2

Поработаем над выделенным состоянием нашего `label::before` или «псевдочекбокса».

Псевдочекбокс при выделении давайте преобразуем в «галочку» — сплющенный блок бирюзового цвета без верхней и правой рамок.

Для управления выделенным состоянием псевдочекбокса создадим CSS-правило с таким селектором:

```css
input[type="checkbox"]:checked ~ label::before { ... }
```
