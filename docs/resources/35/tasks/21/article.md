# Эко-рамка Кексика

У прошлой графической рамки был задан режим повторения `stretch`, который задаётся по умолчанию. На сторонах рамки не было видно сильных искажений при растяжении исходной картинки, так как она имела большие размеры сама по себе.

Картинка для графической рамки в этом задании небольшая и имеет более неравномерный узор, чем золотая рамка предыдущего задания. Давайте возьмём следующие её части для заполнения рамки:

![Пример картинки для рамки](/resources/35/assets/img/eco-frame-example.png)

С такой картинкой будут хорошо видны режимы повторения графической рамки: при растяжении стороны картинки существенно искажаются, поэтому мы попробуем разные значения повторения, чтобы найти оптимальный.
