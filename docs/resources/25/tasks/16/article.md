# Теневое искусство – 3

Подготовка к созданию картины завершена.

У нас есть класс, который создает красивое кольцо с помощью теней. В исходном коде также добавлены элементы и стили, чтобы расположить кольца так, как нам надо.

Вам остаётся только добавить класс к четырем элементам и получится картинка. Напомним, что несколько классов задаются элементу через пробел:

```html
<div class="one two">
```
