# Псевдоклассы `:in-range` и `:out-of-range`

В [части](/chapters/13), посвященной формам, мы разбираем специальный тип поля для ввода числовых значений `<input type="number">`. У этого поля можно определить максимальное и минимальное значение при помощи атрибутов `max` и `min` соответственно.

Селектор `:in-range` выбирает все элементы, значение которых попадает в указанный диапазон.

А селектор `:out-of-range` выбирает все элементы, значение которых НЕ попадает в указанный диапазон.

Например:

```css
input:in-range {}
```
